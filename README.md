# Sales Analysis Excel Project
Sales analysis project using Excel with pivot tables and charts.

## Project Overview

This project focuses on analyzing sales data using Microsoft Excel. The objective of the analysis is to identify key sales trends, understand product performance, and extract insights that could support better business decisions.

The project demonstrates how Excel can be used as a data analysis tool through data cleaning, pivot tables, and visualizations.

## Dataset

The dataset contains sales transaction data including information such as:

* Order date
* Product category
* Sales revenue
* Quantity sold
* Customer segments
* Regional sales performance

## Tools Used

* Microsoft Excel
* Pivot Tables
* Data Cleaning Techniques
* Charts and Data Visualization
* Basic Sales Trend Analysis

## Analysis Performed

Several analytical steps were completed during the project:

* Cleaning and organizing raw sales data
* Creating pivot tables to summarize key metrics
* Analyzing sales performance across different regions
* Identifying top-performing products
* Examining monthly sales trends
* Creating charts and dashboards to visualize insights

## Key Insights

* Certain product categories generate a significant portion of total revenue.
* Sales performance differs across regions and product types.
* Monthly sales trends highlight potential seasonal patterns.
* Some products consistently outperform others in terms of revenue.

## Files Included

* `sales_data.xlsx` – main Excel analysis file
* `dashboard.png` – screenshot of the Excel dashboard
* `charts.png` – visualizations used in the analysis

## Conclusion

This project highlights how Excel can be effectively used for business data analysis and reporting. The insights derived from the dataset demonstrate how data can help support strategic sales decisions.
